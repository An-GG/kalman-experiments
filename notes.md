

# Kalman Filters

On each loop, calc:
-kalman gain
-current estimate
-new error in the estimate

## TERMS

KG : Kalman Gain
ER_EST : Error in estimate
ER_MEA : Error in measurement


## GAIN
"place relative importance in the error in data vs error in estimate"

KG = E_EST / (E_EST + E_MEA)
